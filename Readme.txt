      广播电视大学纸质考试通知单与报考编排明细表合并工具 V1.0
======================================================================
1. 如解压后文件夹名称为中文，请手工修改为英文名称
2. 程序内包含的“省开.xls”文件为要求格式，请参照设置
3. 程序使用方法：
    使用方法一：将包含考试通知单和省开、统设、计算机考生报考编排明细表
的文件夹拖动到本程序上方，然后松手回答问题即可。
    使用方法二：直接运行程序，将要求输入通知单所在文件夹位置。
4. 注意事项：
　　1) 考生报考编排明细表表头必须包含字段：考场号、考试日期、考试时间、
科目(仅代码)、学号、座位号
　　2) 考生报考编排明细表表头字段名空白将被跳过
5. 祝您使用愉快
